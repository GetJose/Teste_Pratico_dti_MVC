#DTI <h1>
**Sistema de Criação de lembrete seguindo o desafio pratico dti**
Projeto desenvolvido no Visual studio, foi se ultilizado no projeto ASP.Net com o SDK 6.0.
##Premissas assumidas<h3>
Que o usuario preenchera os dados corretamente.
##Decisões de projeto<h3>
ultilizar uma lista estática no lugar de um banco de dados, e uma variável "cont" também estática para criar o ID de cada lembrete
